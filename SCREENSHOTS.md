## How to Take Screenshots

### For Desktop/Laptop:
1. Open the application in your browser at `http://localhost:5173`
2. Navigate to each page/view
3. Use your system's screenshot tool:
   - Windows: Windows Key + Shift + S
   - Mac: Command + Shift + 4
   - Linux: PrtScn or Screenshot tool

### For Mobile View:
1. Open browser developer tools (F12)
2. Click the device toolbar icon (or Ctrl+Shift+M)
3. Select a mobile device (e.g., iPhone 12, Pixel 5)
4. Take screenshots of the responsive layout

### Tips:
- Use full-page screenshots when possible
- Ensure the browser window is maximized for desktop views
- Include some sample data for better demonstration
- Capture both light and interaction states (hover, active)
- Show different task statuses and priorities for variety
